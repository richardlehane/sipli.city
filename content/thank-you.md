# Thank you!

Thank you for purchasing a Siplicity license.

You will receive an email from me **within 24 hours** with your license key and license file. These can be added in Siplicity on the settings page (the cog icon on the top right).

As a licensed user, you are entitled to access Siplicity's source code. If you have a Github account, I can enable this access by adding you to the <a href="https://github.com/sipli-city">Siplicity organisation</a>. Please email me at <a href="mailto:richard.lehane@gmail.com">richard.lehane@gmail.com</a> with your Github user name. Or, if you prefer, I'm also happy to send occasional archives of the source code in response to email requests.

Please don't hesitate to reach out to me if you need any help or to provide feedback.

Thanks again
