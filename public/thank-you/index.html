Thank you!

Thank you for purchasing a Siplicity license.

You will receive an email from me within 24 hours with your license key and license file. These can be added in Siplicity on the settings page (the cog icon on the top right).

If you haven’t installed Siplicity yet, it is available for download in the “Try it” section of the Get Siplicity page. Please let me know if you enounter any issues setting it up.

As a licensed user, you have access to Siplicity’s source code. If you have a Github account, email it to me and I can enable this by adding you to the Siplicity organisation. Alternatively, I’m happy to send occasional archives of the source code in response to email requests.

Please don’t hesitate to reach out to me with feedback or if you need any help.
